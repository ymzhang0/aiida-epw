Welcome to aiida-epw's documentation!
================================================

`aiida-epw` is an AiiDA plugin designed to automate and manage high-throughput calculations for EPW software.
It provides a series of workflows for calculating the electron-phonon coupling, superconductivity, and transport properties.

This plugin is built upon the powerful infrastructure of `aiida-quantumespresso` and `aiida-wannier90-workflows`.
